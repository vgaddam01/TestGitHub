Release Notes (English only)

Full details of the monthly release, English only, DOC and PDF format

NOTE: Release Notes are posted at least three times for each release:

  • The Preview version is posted two weeks before the release.

  • The Draft version is posted one week before the release.
  • The Final version is posted the day of the release.

Visit the release notes page often to ensure you have the most current release information.

Release Calendar (English only)

Shows the exact release dates plus the dates the Preview, Draft, and Final versions of the release notes are posted.

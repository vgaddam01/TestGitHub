Delete a recurring payment request

You can delete a request for a particular month or permanently.

To delete a recurring payment request:

  1. On the Payment Request List page, open the recurring payment request by double-clicking it.

  2. Click Actions > Delete Recurring Request.

  3. In the Please Confirm window that appears, click Yes.

A payment request deleted in error can be restored by the AP User role. The request can have been deleted by any role - only the AP User however may restore the request to an active state.

To end a recurring payment request sequence:
  1. Click Invoice > View > Recurring Requests.
  2. On the Recurring Requests page that appears, select the check box of the payment request for which to end the recurring payment request sequence.
  3. In the Actions menu, click Edit Recurring Request. The Recurring Request Details window appears.
  4. Select the end date to the date for when you want the recurring payment request sequence to end.
  5. Click Save.
    Note

    When you end a recurring payment request sequence, no more recurring payment requests will be created. However, the ones already in the system will remain. To remove these, follow the To delete a recurring payment request procedure.

Travel Request preferences

Note Depending on your company's configuration, some of these options may not be available to you. Check with your Concur administrator.

To use these options:

  1. Click Profile > Profile Settings > Request Preferences (in the Request Settings section of the left-side menu).

  2. Make the desired choices.

    Send email when...

    Description

    The status of a request changes

    If selected, an email is sent to an employee when the status of a submitted request has changed. The only exception is when an approver sends the request back to the employee for resubmission. When this happens, a status email is not sent, but a different "Send Back to Employee" email is sent.

    A request is submitted for approval

    If selected, an email is sent to a request approver when a request is made by one of the employees in the approver's group(s).

    Prompt...

    Description

    For an approver when a request is submitted

    If selected, when you submit a request, you are prompted to select a request approver. If you have a default approver, that person's name appears automatically but you can select another approver if desired.

  3. Click Save.

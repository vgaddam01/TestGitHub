January 2017 Mobile Admin Summary

iPhone ONLY – Allocations

Create, Edit, and Delete Allocations

With the 9.41 release, iPhone users can allocate an expense by manually selecting the allocation percentage and cost center. With the expense open, the user taps Allocations to get started. Refer to the release notes for complete details.

Operating System Update

iPhone / iPad

Release 9.41 will be the final release for version 8.0 of the iPhone / iPad operating system.

Before the next release, please ensure that all user device operating systems have been updated to 9.0.

Android

Release 9.41 will be the final release for version 4.1 of the Android operating system.

Before the next release, please ensure that all user device operating systems have been updated to 4.4.

Additional Changes

iPhone / iPad

Improved prompts and user experience when adding a receipt from the Photo Gallery

Minor issue fixes, including those involving PIN reset and the default date on manually entered expenses

Android

Minor issue fixes, including those involving PIN reset

November 2016 Mobile Admin Summary

Android – Updated Home Screen and Main Menu

We have updated the home screen for a cleaner look-and-feel. We have also eliminated the menu in the upper-right corner of the home screen and moved the options (like Settings) to one single menu.

iPhone / iPad – Receipt Image Available on Manual Expense Entry Screen

The receipt image can now be previewed on the screen used to manually enter expenses.

iPhone / iPad – Combined Expenses and Receipts

The new expense list combines receipts and expenses into a single list view. With the combination of receipts and expenses in a single list, spend management and identifying Smart Expenses is much easier.

Android and iPhone / iPad – Fixed Travel Allowances

Users can now claim their fixed meals and fixed lodging travel allowances on mobile.

Fixed vs Reimbursable Travel Allowances: Fixed travel allowances – often referred to as per diems – provide a defined daily amount regardless of the actual amount spent by the user. Reimbursable travel allowances generally provide reimbursement for the actual amount of the expense. The ability to create and manage reimbursable travel allowances is not yet available on mobile. For more information about travel allowances, refer to the Expense: Travel Allowance Setup Guide.

Restrictions: For the most part, fixed travel allowances work the same way on mobile as on the web version of Concur – with some exceptions. These configuration options are not available on mobile:

Users cannot define/select: Trip length, "Short distance", "Extended trips", "Use Percent Rule", Location "within municipality"

Users cannot enter: Actual meal amounts, Rate location

The mobile version does not combine meals and lodging rates nor does it display base rates, company rates, government rates, etc.

If the user's configuration uses any of the options listed above, the user should manage their travel allowances using the web version of Concur.

Approval View: Approvers open the desired report as usual and can view – but not edit – travel allowances

iPhone /iPad – Receipt Image Crop (Ver 9.37)

When 9.37 was released, this feature was released for receipt images. Concur automatically crops a receipt image to remove the background – resulting in only the receipt image displaying. The feature is on by default but can be turned off by the user. If the user chooses to disable the cropping process, the user can tap the icon in the lower-right corner of the screen.

Android and iPhone / iPad – Blurred Receipt Image (Ver 9.37)

When 9.37 was released, this feature was released for receipt images. If Mobile detects that the receipt image is blurry, this message appears in the lower-left corner of the screen: Blurry image detected. Please retake.

The user should consider getting a better image, which may ensure more timely reimbursement of the associated expense.
